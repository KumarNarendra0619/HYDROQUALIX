## 🌍 Overview

**HydroQualix-4™** is a web-based geospatial Water Quality Index (WQI) analysis system designed for environmental monitoring, hydrological research, and data-driven decision-making.

It integrates GIS concepts with scientific WQI modeling to visualize and evaluate water quality across spatial locations.
